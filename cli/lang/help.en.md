# Keyboard Commands:

    A-Z          :  add a letter to the tray
    
    < >          :  change board location of current letter
    
    -            :  remove the current letter from the tray
    
    (space)      :  clear the tray
    
    (enter)      :  play the word currently in the tray
    
    (Control-C)  :  quit game
