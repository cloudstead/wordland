# Keyboard Commands:

    A-Z          :  add a letter to the tray
    
    < >          :  change tile location of current letter
    
    -            :  remove current letter from the tray
    
    (space)      :  clear the tray
    
    (enter)      :  play the word currently in the tray
    
    /            :  list all games in progress
    
    (Control-C)  :  quit game
